Como é uma estrutura básica de um HTML?

Um documento HTML começa com a tag <!DOCTYPE html>, que informa ao navegador que estamos utilizando HTML5.

Em seguida, temos a estrutura geral do código:

  1. <html>: envolve todo o conteúdo da página.
  2. <head>: contém informações sobre a página, como o título e os metadados.
  3. <body>: parte visível da página, onde colocamos textos, imagens, links etc.

Aqui está um exemplo completo de uma estrutura HTML básica:

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
  </body>
</html>

Dica: O ideal é sempre manter seu HTML semântico e organizado para facilitar a manutenção e o entendimento.

Entendendo a diferença entre listas ordenadas e não ordenadas

As listas ordenadas usam a tag <ol> e mostram os itens numerados:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

As listas não ordenadas usam a tag <ul> e mostram marcadores:

  • Item qualquer
  • Outro item
  • Mais um item
